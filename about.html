The Story Behind the Site

Hogwarts Houses & Wizards celebrates the spirit of the series with respectful nods to canon, careful spoiler handling, and an accessible design crafted for fans old and new.

About This Project

Project Description

This mini-site reimagines a multi-page template as a magical hub for Hogwarts lore. The experience highlights each house’s culture, showcases notable witches and wizards with spoiler-aware summaries, and strings major events into a living timeline you can explore.

Built using AI prompts only — no manual edits. Every card, modal, and timeline marker is powered by structured data so content stays consistent across the site.
